Simulates one time step of request handling across all web servers. 


\begin{DoxyItemize}
\item Idle servers take new requests from the queue.
\item Servers completing their work return to idle state.
\item Idle servers with no requests may be removed.
\item A new server may be added if the load is high.
\item Occasionally generates a new random request.
\end{DoxyItemize}
